SCRAP N' SWAP

Roles

UI Designer and UX Designer (Solo)

Project Summary

This project was focused on UI and UX design for a couch coop game. I concepted a casual competitive game where up to four players take turns completing objectives to see who can score the most points. It features a hot swap mechanic where players pass the controller between themselves when it is their turn.

Workflow

I completed a lot of research for this project before starting to design the interface. I focused on accessibility to try make my interface useable by as large an audience as possible.


I started with paper prototype sketches to get a rough idea of how the interface would look before moving into Pencil to create a digital prototype that could be iterated on.


Finally an interactive prototype was made in Adobe XD to see how the whole interface would function in a game.

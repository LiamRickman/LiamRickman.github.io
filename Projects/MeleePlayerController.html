MELEE PLAYER CONTROLLER

Roles

Designer and Programmer (Solo)

Project Summary

This project saw the creation a third-person player controller that focuses on melee combat and advanced movement mechanics.


I built the player controller in Unreal Engine which started out basic with standard movements. I then iterated upon the controller until it featured dodging, vaulting, and climbing movement.


It also has melee attacks using a thermal katana, this can be used with the players ultimate which enhances their attacks temporarily. To showcase the combat more appropriately I also created some enemy AI for the player to attack.

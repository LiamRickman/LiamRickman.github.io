Roles

Designer, Programmer, and Data Analyst (Solo)

Project Summary

UFO Resistance is a 2D tower defence game that I created for a data collection and data-driven design project.


I first programmed the tower defence game and then ran playtests where I collected quantitative and qualitative data. This was analysed and then used to create new improved iterations of the game.

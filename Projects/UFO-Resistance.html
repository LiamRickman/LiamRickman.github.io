UFO Resistance

Development: 
2021-22 (Solo)

Platform: 
PC

Software Used: 
Unity 2019.4.19
UFO Resistance  is a tower defence game I created as part of a data collection and data-driven design project.

As a primarily data driven project, I quickly prototyped a basic project with the help of some online tutorials. I ran multiple playtests to collect data and then iterated on the prototype based on the data collected.

I collected both quantitative and qualititive data as part of this project to work with both statistics and feedback to improve the final prototype.

Iterations Video

Turret Mechanics

As a tower defence game, the primary mechanic is placing turrets to stop enemies reaching the end base. I gave players three choices for turrets: Standard, Laser, and Rocket. These have various costs and stats to give players some variation.

I also added an upgrade mechanic to allow players to improve the turrets over time. This uses upgrade tokens instead of the regular currency to encourage players to upgrade turrets instead of only buying new ones.

Level Variations

While this was only a prototype project, I still included five levels and a tutorial level so that players can get a good experience and provide useful data across a variety of levels. 

This helped when collecting data as I could determine if the level layout had any impact on how players experienced the game.

SLIPPING MINDS

Role

Lead Programmer (Team: 4)

Project Summary

Slipping Minds was a game jam project created in 24 hours over two days. Working in a team of four we created a game based around memory loss. The game is played from the perspective of a father who is travelling looking for his lost son.


Over the course of the game he begins to get tired and loses his memory as shown with the game getting darker and more greyscale.


The game uses a platformer layout with some key mechanics including, grappling hook swings, spikes and saws

Year

2019

Platform

PC

Software Used

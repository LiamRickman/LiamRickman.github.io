Project Title

Slipping Minds was a game jam project created in 24 hours over two days. Working in a team of four we created a game based around memory loss. The game is played from the perspective of a father who is looking for his lost son.

Over the course of the game he begins to get tired and loses his memory as shown with the game getting darker and more greyscale. The game uses a platformer layout with some key mechanics including a grappling hook, spikes and saws.

For this game jam we were awarded the best overall game prize.
